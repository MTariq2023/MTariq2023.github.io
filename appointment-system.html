Appointment Booking System

This is a desktop-based C# Windows Forms application that allows users to schedule, confirm, and view medical appointments. It uses a PostgreSQL database and Npgsql for database interaction. Key features include form input validation, confirmation dialogs, and a DataGridView to display appointment records.

Tech Stack: C#, .NET, Windows Forms, PostgreSQL, Npgsql
