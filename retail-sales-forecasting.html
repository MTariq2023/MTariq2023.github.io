Analyzing and Visualizing Real vs Nominal U.S. Retail Sales Trends

Using Python and Tableau

Project Overview

This project cleans, merges, and analyzes U.S. retail sales and inflation data to compare nominal vs. real trends over time. The processed dataset was visualized in Tableau to highlight the effect of inflation on retail spending patterns.

Tech Stack: Python (Pandas), Tableau, CSV/XLSX
